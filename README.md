# 📱 Daily Notes App

Aplikasi mobile berbasis React Native untuk mencatat aktivitas harian pengguna.

---

## ✨ Fitur

* Tambah catatan
* Edit catatan
* Hapus catatan
* Penyimpanan lokal

---

## 🛠️ Teknologi

* React Native
* JavaScript
* AsyncStorage

---

## 🚀 Cara Menjalankan

```bash
npm install
npx expo start
```

---

## 📌 Tujuan Project

Project ini dibuat untuk melatih pengembangan aplikasi mobile dan manajemen data lokal.
